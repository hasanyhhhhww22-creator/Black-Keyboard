# BLACK Keyboard - لوحة مفاتيح أندرويد ذكية

أهلاً بك في مشروع "BLACK Keyboard"! هذا هو الكود المصدري الكامل للوحة مفاتيح أندرويد ذكية ومخصصة، تم تطويرها خصيصاً لك.

**الميزات الرئيسية:**
1.  **الثيمات والتخصيص الكامل:** تغيير ألوان الخلفية والأزرار، دعم الصور الشخصية كخلفية.
2.  **الذكاء الاصطناعي لتوليد النصوص (Gemini AI):** زر خاص يقترح إكمالات أو طرق مختلفة لصياغة الجملة (رسمي، غير رسمي، عاطفي).
3.  **الزخرفة والرموز:** مكتبة داخلية تحتوي على زخارف ورموز جاهزة.
4.  **الاختصارات الذكية (Custom Rules):** تعريف اختصارات نصية مخصصة (مثلاً: "بس" -> "بسم الله الرحمن الرحيم").
5.  **الخصوصية:** لا يجمع بيانات الكتابة، وخاصية تصفية المحتوى الحساس قابلة للتفعيل.

## كيفية الحصول على ملف APK الخاص بك (باستخدام GitHub Actions)

بما أنك تملك حساب GitHub، يمكنك بسهولة تحويل هذا الكود إلى ملف APK قابل للتثبيت على هاتفك دون الحاجة لتثبيت Android Studio.

### الخطوة 1: إنشاء مستودع (Repository) جديد على GitHub
1.  اذهب إلى [GitHub](https://github.com/) وسجل الدخول.
2.  اضغط على زر `+` في الزاوية العلوية اليمنى، ثم اختر `New repository`.
3.  أدخل اسم المستودع (مثلاً: `BLACK-Keyboard-Android`).
4.  اجعل المستودع `Public` أو `Private` حسب رغبتك.
5.  **لا** تحدد خيار `Add a README file` أو `Add .gitignore` أو `Choose a license` في هذه المرحلة.
6.  اضغط على `Create repository`.

### الخطوة 2: رفع ملفات المشروع إلى المستودع الخاص بك

الآن سأقوم بتزويدك بملف مضغوط (`.zip`) يحتوي على جميع ملفات المشروع. بعد تحميله:

1.  قم بفك ضغط الملف على جهاز الكمبيوتر الخاص بك (أو يمكنك استخدام تطبيق لفك الضغط على هاتفك).
2.  داخل المستودع الجديد الذي أنشأته على GitHub، ستجد خيار `uploading an existing file` أو `Add file` ثم `Upload files`.
3.  قم بسحب وإفلات جميع المجلدات والملفات الموجودة داخل المجلد الرئيسي `BLACK_Keyboard_Keyboard_Keyboard_Keyboard_Keyboard_Keyboard_Keyboard_Keyboard` (بما في ذلك المجلدات المخفية مثل `.github`) إلى صفحة الرفع على GitHub.
4.  بعد رفع جميع الملفات، اضغط على زر `Commit changes`.

### الخطوة 3: تشغيل عملية البناء (Build) على GitHub Actions

بمجرد رفع الملفات، سيبدأ GitHub Actions تلقائياً في بناء التطبيق:

1.  في مستودعك على GitHub، اذهب إلى تبويب `Actions`.
2.  سترى workflow باسم `Android Release Build`.
3.  اضغط عليه، وسترى عملية بناء جارية. قد تستغرق بضع دقائق.
4.  عندما تكتمل العملية بنجاح، سترى علامة صح خضراء.

### الخطوة 4: تحميل ملف APK

1.  بعد اكتمال عملية البناء، اضغط على `Android Release Build` ثم على `build` في القائمة اليسرى.
2.  في قسم `Artifacts`، ستجد ملفاً باسم `app-release`.
3.  اضغط على `app-release` لتحميل ملف `app-release.apk` إلى جهازك.

### الخطوة 5: تثبيت لوحة المفاتيح وتفعيلها

1.  انقل ملف `app-release.apk` إلى هاتفك وقم بتثبيته. قد تحتاج إلى تفعيل خيار "تثبيت التطبيقات من مصادر غير معروفة" في إعدادات هاتفك.
2.  بعد التثبيت، اذهب إلى `إعدادات الهاتف` -> `النظام` -> `اللغات والإدخال` -> `لوحات المفاتيح الافتراضية` (قد تختلف التسميات قليلاً حسب إصدار أندرويد).
3.  قم بتفعيل "BLACK Keyboard".
4.  اختر "BLACK Keyboard" كلوحة مفاتيح افتراضية.

## إعدادات هامة داخل التطبيق

بعد تفعيل لوحة المفاتيح، يمكنك فتح تطبيق "BLACK Keyboard" من قائمة التطبيقات للوصول إلى الإعدادات:

1.  **Gemini API Key:** ادخل مفتاح Gemini API الخاص بك في قسم "AI Settings" لتفعيل ميزة الذكاء الاصطناعي.
2.  **تصفية المحتوى (Content Filter):** يمكنك تفعيلها من قسم "Privacy Settings". لتعديل الكلمات الحساسة، ستحتاج إلى تعديل الكود المصدري في ملف `BlackKeyboardService.kt` (ابحث عن `sensitiveWords`).
3.  **الثيمات:** يمكنك تغيير الألوان واختيار صورة خلفية من معرض الصور الخاص بك.
4.  **الاختصارات والقواعد المخصصة:** يمكنك إضافة اختصاراتك الخاصة من قسم "Custom Rules".
5.  **مكتبة الزخرفة:** تصفح وانسخ الزخارف والرموز الجاهزة.
